Project 2 - Spaceship Interface

This is the circuit from the second tutorial in the Arduino Projects Book from the official starter kit from Arduino.cc.

The purpose of this project is to discover digital inputs and outputs, understand how Arduino sketches work, and the use of LEDs, Resistors, and pushbuttons

Parts required:
  • 1 green LED
  • 2 red LEDs
  • pushbutton
  • 10 kilohm resistor
  • 3 220 ohm resistors

                        
The Code

// the variable switchState and initialize it by the value 0
int switchState = 0;

// the setup function runs once when you press reset or power the board 
void setup() {
    // initialize digital pin LED_BUILTIN as an output. 
    pinMode(2, INPUT);
    pinMode(3, OUTPUT); // green light 
    pinMode(4, OUTPUT); // red light 
    pinMode(5, OUTPUT); // red light 
    }
                            
//  the loop function runs over and over again forever
void loop() {
    switchState = digitalRead(2); //read-in the switch status from pin2
                                
    if (switchState == LOW){  button is not pressed 
                                
    digitalWrite(3, HIGH);  // green light is on 
    digitalWrite(4, LOW);   //red light1 is off 
    digitalWrite(5, LOW);   //red light2 is off 
    }
                                
    else {  button is pressed 
                                
    digitalWrite(3, LOW);   // green light is off  
    digitalWrite(4, LOW);   // red light1 is off  
    digitalWrite(5, HIGH);  // red light2 is on  
                                
    delay(500); //  wait for quarter second(250ms)  
    digitalWrite(4, HIGH);  //  red light1 is on  
    digitalWrite(5, LOW);   //  red light2 is off  
    delay(250); //  wait for a quarter second(250ms)  
                                
    } // go back to the beginning of the loop  
                                
};

Sketch uses 1242 bytes (3%) of program storage space. Maximum is 32256 bytes.
Global variables use 11 bytes (0%) of dynamic memory, leaving 2037 bytes for local variables. Maximum is 2048 bytes.
